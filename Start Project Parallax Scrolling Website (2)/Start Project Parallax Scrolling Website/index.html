Características Genéticas Dominantes e Recessivas

Características Dominantes: Estas são expressas fenotipicamente mesmo quando apenas uma cópia do gene (alelo) responsável está presente. Em outras palavras, se uma pessoa herda um alelo dominante de um dos pais, essa característica será visível. Por exemplo, a presença de sardas e a capacidade de enrolar a língua são geralmente características dominantes.

Características Recessivas: Estas só são expressas fenotipicamente quando uma pessoa herda duas cópias do gene recessivo (um de cada pai). Se uma pessoa tem um alelo recessivo e um dominante para uma característica, a característica dominante será a expressa. Exemplos de características recessivas incluem a cor dos olhos azuis e o lóbulo da orelha unido.

Características:

Formato de Rosto:

Rosto oval

Dominante rosto oval é caracterizado por uma forma mais alongada, com linhas suaves e uma testa mais larga do que o queixo. Neste contexto genético, "dominante" significa que, se uma pessoa herdar pelo menos um alelo para o rosto oval, essa característica será expressa, independentemente de quaisquer outros alelos presentes.

Rosto quadrado

Recessivo Um rosto quadrado tem uma estrutura mais angular, com linhas retas nas bochechas e maxilar. Aqui, "recessivo" indica que esta característica só será expressa quando uma pessoa herda dois alelos para o rosto quadrado, um de cada progenitor.

Presença de sardas :

Presença de sardas

Dominante Se uma pessoa herda pelo menos um alelo para a presença de sardas, essa característica será expressa. Isso significa que mesmo se a pessoa herdar um alelo para sardas e um alelo para a ausência de sardas, as sardas serão visíveis. Portanto, a presença de sardas é considerada uma característica dominante.

Sardas ausentes

Recessivo Para a ausência de sardas ser expressa, é necessário que uma pessoa herde dois alelos recessivos para essa característica, um de cada progenitor. Se a pessoa herdar pelo menos um alelo dominante para sardas, essa característica será expressa, mascarando a característica recessiva para a ausência de sardas.

Polidactilia:

Polidactilia

Dominante Se uma pessoa herdar pelo menos um alelo para a polidactilia, essa característica será expressa. Mesmo que a pessoa herde um alelo para polidactilia e um alelo para a ausência de polidactilia, os dedos extras serão visíveis. Portanto, a presença de polidactilia é considerada uma característica dominante.

Polidactilia ausente

Recessivo Para a ausência de polidactilia ser expressa, é necessário que uma pessoa herde dois alelos recessivos para essa característica, um de cada progenitor. Se a pessoa herdar pelo menos um alelo dominante para polidactilia, essa característica será expressa, mascarando a característica recessiva para a ausência de polidactilia.

Lateralidade:

Destro

Dominante Se uma pessoa herda pelo menos um alelo para destro, essa característica será expressa. Isso significa que mesmo que a pessoa herde um alelo para destro e um alelo para canhoto, ela será destro. Portanto, ser destro é considerado uma característica dominante.

Canhoto

Recessivo Para ser canhoto, uma pessoa deve herdar dois alelos recessivos para essa característica, um de cada progenitor. Se a pessoa herdar pelo menos um alelo dominante para destro, ela será destro, mascarando a característica recessiva para canhoto.
